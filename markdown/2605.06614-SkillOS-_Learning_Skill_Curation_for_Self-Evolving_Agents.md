## Abstract

LLM-based agents increasingly handle streaming tasks yet often remain one-off problem solvers. We propose SkillOS, an experience-driven RL training recipe for learning skill curation in self-evolving agents. SkillOS pairs a frozen agent executor that retrieves and applies skills with a trainable skill curator that updates an external SkillRepo from accumulated experience. Across multi-turn agentic tasks and single-turn reasoning tasks, SkillOS consistently outperforms memory-free and strong memory-based baselines. The learned curator produces more targeted skill use, while skills evolve into richly structured Markdown files encoding higher-level meta-skills.
