## Abstract

LLM-based workflows compose specialized agents to execute complex tasks, and these agents usually share substantial context, allowing KV-Cache reuse. We build PBKV (Prediction-Based KV-Cache Management). For each workflow, PBKV predicts agent invocations in future steps by fusing guidance from historical workflows and target workflow context, estimates reuse potential of cache entries, and keeps high-potential entries in GPU memory. Experiments show PBKV achieves up to 1.85x speedup over LRU on dynamic workflows, and up to 1.26x speedup over KVFlow on static workflows.
