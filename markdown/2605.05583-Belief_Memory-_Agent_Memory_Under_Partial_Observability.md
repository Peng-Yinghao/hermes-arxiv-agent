## Abstract

LLM agents that operate over long context depend on external memory to accumulate knowledge. Existing methods store each observation as a single deterministic conclusion, introducing self-reinforcing error by discarding uncertainty. We propose BeliefMem, which shifts the memory paradigm to retaining multiple candidate conclusions with probabilities updated via Noisy-OR rules. At retrieval, all candidates surface with probabilities, keeping alternatives visible to the agent. Empirical evaluations on LoCoMo and ALFWorld show BeliefMem achieves the best average performance, remarkably outperforming well-known baselines. Probabilistic memory produces substantial gains and explores a new direction for agent memory in partially observable environments.
