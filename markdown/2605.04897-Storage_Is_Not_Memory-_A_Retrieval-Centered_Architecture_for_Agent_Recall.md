## Abstract

We propose True Memory, a six-layer architecture that shifts the center of the system from a storage schema to a multi-stage retrieval pipeline operating over events preserved verbatim. The full system runs as a single SQLite file on commodity CPU. On LoCoMo (1,540 questions), True Memory Pro reaches 93.0% accuracy against 61.4% for Mem0, 65.4% for Supermemory, ~71% for Zep, and 94.5% for EverMemOS. On LongMemEval, True Memory Pro reaches 87.8%. On BEAM-1M (1M token scale), True Memory Pro reaches 76.6%, above prior published result of 73.9% for Hindsight.
