## Abstract

We present Code Broker, a multi agent system built on Google's ADK that analyses Python source code and generates structured quality assessment reports. The system uses a hierarchical five agent architecture: a root orchestrator coordinates a sequential pipeline that dispatches three specialised agents concurrently (Correctness Assessor, Style Assessor, Description Generator). Reports quantify correctness, security, style, and maintainability. Code Broker fuses LLM-based semantic reasoning with Pylint signals, uses async execution with exponential backoff, and explores lightweight session memory for cross-run context.
