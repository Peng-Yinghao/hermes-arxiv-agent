## Abstract

In agent memory systems, generic reranking models rely on semantic similarity matching and lack genuine reasoning capabilities, leading to semantically relevant but unhelpful results. This report introduces MemReranker, a reranking model family (0.6B/4B) built on Qwen3-Reranker through multi-stage LLM knowledge distillation. Training data combines general corpora with memory-specific multi-turn dialogue data covering temporal constraints, causal reasoning, and coreference resolution. MemReranker-0.6B substantially outperforms BGE-Reranker and matches open-source 4B/8B models. MemReranker-4B achieves 0.737 MAP, on par with Gemini-3-Flash on key metrics, at only 10-20% of inference latency.
